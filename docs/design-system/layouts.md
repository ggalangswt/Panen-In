# Layouts And Cards

Sumber: Figma `node-id=76:753`

## Guide Section Layout

Label section:
- teks uppercase/section label menggunakan `15px / 22.5px / 500`
- warna `neutral.ne30`

Container stack:
- layout vertikal
- gap antar card `10px`

## Guide Card Pattern

Spesifikasi:
- background putih
- border `1px solid neutral.ne20`
- radius `20px`
- padding horizontal `20px`
- padding vertical `15px`
- gap internal utama `15px`
- lebar referensi `386px`

Header card:
- kiri: judul tahapan bernomor
- kanan: chip waktu/status

Title style:
- `15px / 22.5px / 700`
- warna `neutral.ne50`

Chip style:
- background `alert.al00`
- border `primary.pr10`
- radius pill
- padding horizontal `10px`
- padding vertical `4px`
- teks `12px / 18px / 500`
- warna `primary.pr30`

Body copy:
- `14px / 21px / 400`
- warna `neutral.ne30`
- antar baris poin memakai gap `4px`

## Reusable Pattern Notes

- Kombinasi `title + chip + list body` ini sebaiknya dijadikan pola card standar untuk panduan tanam, tahapan, atau edukasi step-by-step.
- Radius `20px` dipakai untuk card konten besar, sedangkan radius `10px` dipakai untuk action kecil, tombol, dan item pilihan.
- Border netral tipis adalah baseline hampir semua card di desain ini.
