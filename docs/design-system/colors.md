## Praktik Implementasi

- Gunakan `primary.pr30` untuk CTA, state aktif, dan brand emphasis.
- Gunakan `primary.pr00` dan `primary.pr10` untuk selected card atau status lembut.
- Gunakan `neutral.ne50` untuk heading dan label utama.
- Gunakan `neutral.ne30` untuk body text dan label sekunder.
- Gunakan `neutral.ne20` sebagai border default kartu, nav, dan input.
